# 项目简介

基于深度学习YOLOv11的水下鱼类检测系统，致力于提升海洋生物识别效率。系统功能包括实时鱼类检测、种类识别及数据记录，采用YOLOv11算法提升检测精度与速度。技术特点在于运用自定义YOLO数据集进行模型训练，配备用户友好的UI界面以及登录注册功能，确保使用便捷性与数据安全性。本项目提供完整的源代码，助力水下观察与研究工作。


## 核心功能
- 实时鱼类检测
- 种类识别
- 数据记录
- 用户友好的UI界面
- 登录注册功能
- 完整源代码提供

## 适用场景
- 水下观察与研究工作
- 海洋生物识别
- 水域生态监测
- 科研数据收集
- 海洋生物多样性研究

## 技术特点
- 基于YOLOv11深度学习算法
- 自定义YOLO数据集进行模型训练
- 提升检测精度与速度
- 配备登录注册功能确保数据安全性
- 完整源码便于二次开发与部署
